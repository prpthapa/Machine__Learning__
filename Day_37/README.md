## Day 37 - From Business Problems to Machine Learning Problems

Today I learned about the real-world workflow followed by Data Scientists when solving business problems using Machine Learning.

### Topics Covered

- Understanding business problems
- Converting business problems into machine learning problems
- Identifying the type of machine learning problem
  - Classification
  - Regression
- Defining project goals and success metrics
- Feature selection and feature engineering
- Understanding data requirements
- Collaboration with Data Engineers
- Model evaluation metrics
- Online vs Offline Learning
- Decision-making in machine learning projects

### Key Learning

A Data Scientist's job is not just building models. The first step is understanding the business objective and translating it into a measurable machine learning problem.

This session helped me understand the complete thought process behind designing and planning a machine learning solution before writing any code.
