## Day 4 - Advanced NumPy Operations

Today I explored more advanced NumPy concepts related to array manipulation and data selection:

- Reshaping arrays
- Flattening arrays using `ravel()`
- Transposing arrays using `transpose()`
- Combining arrays using `hstack()`
- Splitting arrays using `vsplit()`
- Fancy indexing
- Boolean indexing
- Plotting graphs using NumPy-generated data

These topics improved my understanding of efficient array operations and data visualization.
