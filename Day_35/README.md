### Key Concepts Learned

#### What is a Tensor?
A tensor is a mathematical structure used to represent data in machine learning and deep learning. It is a generalization of scalars, vectors, and matrices to higher dimensions.

#### Types of Tensors

- **0D Tensor (Scalar):** A single value
- **1D Tensor (Vector):** A list of values
- **2D Tensor (Matrix):** Rows and columns of values
- **3D Tensor:** A collection of matrices
- **4D Tensor:** Often used for batches of images
- **5D Tensor:** Commonly used for video data and advanced deep learning applications

This session helped me understand the fundamental data structure used in modern machine learning and deep learning frameworks such as TensorFlow and PyTorch.
