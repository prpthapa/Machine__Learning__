## Day 45 - Feature Scaling: Normalization Techniques

Today I learned about **Normalization**, another important feature scaling technique used in Machine Learning.

### Topics Covered

- What is Normalization?
- Min-Max Scaling (MinMaxScaler)
- Intuition behind Min-Max Scaling
- Code implementation of Min-Max Scaling
- Mean Normalization
- Max Absolute Scaling (MaxAbsScaler)
- Robust Scaling (RobustScaler)
- Normalization vs Standardization

### Key Concepts Learned

- **Normalization** rescales numerical features to a fixed range, typically **0 to 1**.
- **Min-Max Scaling** transforms data while preserving the relationships between values.
- **Mean Normalization** scales data around its mean to center the distribution.
- **MaxAbs Scaling** scales values by dividing by the maximum absolute value, making it suitable for sparse datasets.
- **Robust Scaling** uses the median and interquartile range (IQR), making it less sensitive to outliers.
- Understanding the difference between **Normalization** and **Standardization** helps in selecting the appropriate preprocessing technique for different machine learning algorithms.

This session helped me understand various normalization techniques and when to apply them for better machine learning model performance.
