## Day 39 - Fetching Data Using APIs

Today I learned and practiced:

- Understanding what an API is
- Fetching data from APIs
- Making HTTP requests using Python
- Working with API responses
- Parsing JSON data received from APIs
- Converting API data into Pandas DataFrames
- Exploring real-world datasets through APIs

### Key Concepts Learned

- APIs (Application Programming Interfaces) allow applications to communicate and exchange data.
- Most modern APIs return data in JSON format.
- Python libraries such as `requests` can be used to send HTTP requests and retrieve data.
- API data can be transformed into Pandas DataFrames for analysis and machine learning workflows.

### Example Workflow

1. Send a request to an API endpoint.
2. Receive the response.
3. Extract JSON data.
4. Convert the data into a DataFrame.
5. Perform analysis and preprocessing.

This session helped me understand how real-world data is often collected directly from APIs instead of static files such as CSVs.
