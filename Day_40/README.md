## Day 40 - Web Scraping

Today I learned and practiced:

- Introduction to Web Scraping
- Understanding HTML structure
- Inspecting web pages using browser developer tools
- Extracting data from websites
- Using the `requests` library to fetch web pages
- Using `BeautifulSoup` to parse HTML content
- Finding elements using tags, classes, and IDs
- Extracting and cleaning web data

### Key Concepts Learned

- Web Scraping is the process of automatically collecting data from websites.
- HTML elements can be identified using tags, classes, and IDs.
- The `requests` library is used to retrieve webpage content.
- `BeautifulSoup` helps parse and navigate HTML documents.
- Scraped data can be stored in Pandas DataFrames for analysis.

### Basic Workflow

1. Send a request to a webpage.
2. Retrieve the HTML content.
3. Parse the HTML using BeautifulSoup.
4. Locate the required elements.
5. Extract and clean the data.
6. Store the data for analysis.

This session helped me understand how to collect data directly from websites when datasets are not readily available through files or APIs.
