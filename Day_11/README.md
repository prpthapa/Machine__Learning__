## Day 11 - Data Cleaning and Group Operations

Today I learned and practiced:

- Resetting index using `reset_index()`
- Renaming columns using `rename()`
- Finding unique values with `unique()`
- Counting unique values with `nunique()`
- Detecting missing values using `isnull()` and `notnull()`
- Handling missing data with `dropna()`
- Grouping data using `groupby()`
- Applying custom functions using `apply()`

These concepts helped me improve data cleaning, grouping, and transformation skills using Pandas.
