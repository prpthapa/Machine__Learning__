# Day 1: NumPy Foundations

Today I started learning the foundations of NumPy, an essential library for numerical computing in Python.

## Topics Covered
- Difference between Python lists and NumPy arrays  
- Why NumPy is more efficient for numerical operations  
- Creating arrays using:
  - Zeros arrays (`np.zeros()`)
  - Ones arrays (`np.ones()`)
  - Random arrays (`np.random`)

## Key Learnings
- NumPy arrays are faster and more memory efficient than Python lists.
- Vectorized operations make calculations simpler and more optimized.
- NumPy provides built-in functions to quickly generate structured arrays.
