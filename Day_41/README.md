## Day 41 - Univariate Analysis with the Titanic Dataset

Today I learned and practiced:

- Introduction to Univariate Analysis
- Analyzing categorical data using `countplot()`
- Analyzing numerical data using `histplot()`
- Understanding data distribution with `displot()`
- Detecting outliers using `boxplot()`
- Exploring individual features in the Titanic dataset

### Key Concepts Learned

- **Univariate Analysis** focuses on analyzing a single variable at a time.
- **Count Plot** is used to visualize the frequency of categorical variables.
- **Histogram** shows the distribution of numerical data.
- **Box Plot** helps identify the spread of data and detect outliers.
- **Distribution Plot (`displot`)** provides insights into the shape and density of numerical data.

This session helped me understand how to explore individual features of a dataset before moving to multivariate analysis and machine learning.
