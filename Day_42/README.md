## Day 42 - Bivariate and Multivariate Analysis

Today I learned and practiced:

- Bivariate Analysis
- Multivariate Analysis
- Scatter Plot (Numerical vs Numerical)
- Relational Plot (`relplot`) for multivariate visualization
- Bar Plot (Numerical vs Categorical)
- Box Plot (Numerical vs Categorical)
- Distribution Plot (`kdeplot`) for category comparison
- Heatmap (Categorical vs Categorical)
- Clustermap
- Pair Plot
- Line Plot

### Key Concepts Learned

- **Bivariate Analysis** helps analyze the relationship between two variables.
- **Multivariate Analysis** helps study relationships among multiple variables simultaneously.
- **Scatter plots** reveal relationships between numerical variables.
- **Bar plots and box plots** compare numerical values across categories.
- **KDE plots** compare the distribution of numerical variables across different categories.
- **Heatmaps and clustermaps** visualize relationships between categorical variables.
- **Pair plots** provide pairwise relationships among multiple numerical features.
- **Line plots** help visualize trends over time.

This session strengthened my understanding of exploratory data analysis by learning how different visualization techniques can uncover relationships and patterns within datasets.
