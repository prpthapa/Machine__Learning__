## Day 38 - Loading Datasets from JSON and SQL

Today I learned and practiced:

- Reading JSON files using Pandas
- Loading JSON data from local files and URLs
- Understanding the structure of JSON data
- Connecting to SQL databases
- Reading SQL tables into Pandas DataFrames
- Executing SQL queries and importing the results into Pandas

### Key Concepts Learned

- JSON is a widely used format for storing and exchanging structured data.
- Pandas provides `read_json()` to easily load JSON datasets.
- SQL databases can be connected to Python using libraries such as `sqlite3` or SQLAlchemy.
- Pandas provides `read_sql()` and `read_sql_query()` to import data directly from databases.

This session helped me understand how to work with datasets stored in different formats, making data collection and preprocessing more efficient for data analysis and machine learning projects.
